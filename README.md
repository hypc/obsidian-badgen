# Obsidian Badgen

A [shields badgen](https://shields.io/) plugin for [Obsidian](https://obsidian.md/).

## Examples

![examples](examples.gif)

## Usage

```
!!! badgen github :user/:repo
!!! badgen docker :library
!!! badgen docker :user/:library
!!! badgen npm :package
!!! badgen npm :scope/:package
```
